What We Learned About HTML

How to use block commands

We learned several block commands

  1. HTML defines the type of file and how it is read by the browser/li>
  2. body contains all the content
  3. head contains title and meta data
  4. hx tags indicate headers
  5. p tag indicates a paragraph
  6. ul and ol represent unordered and ordered lists
  7. li is a list item within a ul or ol tag

How to use inline styles

Inline styles are included in the text of the HTML. We learned a handful of inline tags

  1. strong for bold
  2. em for italics
  3. a tag for hyperlinks
  4. img tag for images
