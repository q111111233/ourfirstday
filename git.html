What We Learned About Git

How to create a local repository

We create a local repository by using these commands

  1. create folder
  2. git init within folder
  3. git pair

We created a remote repository

  1. login to github
  2. create new repository on git website
  3. use git remote add nickname url to link remote repository to local repository

We learned how to commit

Committing is done locally and then remotely

Committing locally

  1. git add the modified files
  2. git-pair-commit with a message

Committing remotely

  1. git push nickname master
